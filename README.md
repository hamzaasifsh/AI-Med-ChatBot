## 🚀 About The Project

The **AI Medical Chatbot** is an intelligent healthcare assistant designed to provide basic medical guidance and improve accessibility to health-related information.

It allows users to:
- interact using text and voice  
- get symptom-based responses  
- receive general health suggestions  

⚠️ *This chatbot is for educational purposes only and not a replacement for professional medical advice.*

---

## ✨ Features

- 🧠 AI-based response system  
- 🎤 Voice input (speech recognition)  
- 🔊 Voice output (text-to-speech)  
- 💬 Chat-based interaction  
- ⚡ Fast and responsive system  
- 🩺 Basic symptom understanding
